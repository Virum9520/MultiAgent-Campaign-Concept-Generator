## Campaign Concept Generator using AI Agents.

#### This tool takes a brand + product input, searches current trends via MCP, and uses a multi-agent system to generate 3-5 campaign concepts in minutes.

#### Demo - https://virum9520.github.io/MultiAgent-Campaign-Concept-Generator/

### Tech Stack

##### Built with: Python, FastAPI, LangGraph, LangChain, OpenAI API, Claude, Chroma DB (vector storage), React + Tailwind CSS, MCP (Model Context Protocol) for web search integration

### Architecture highlights:

##### Multi-agent workflow with specialized agents (Trend Research → Strategy → Creative Generation → Quality Validation)

##### Real-time trend discovery using web search MCP

##### Vector DB for campaign examples and pattern matching

##### Quality gates ensuring output relevance, brand alignment, and creative diversity
